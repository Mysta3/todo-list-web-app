## Description

Created from scratch in order to display web development fundamentals across the stack.

## Technologies Used

- ReactJs with Hooks
- SASS
- MongoDB
- ExpressJs
- NodeJs
- Axios
- Firebase Authentication

## Get Started

- Clone & or Download
- Navigate into todo-list
- run `javascript npm i`
- run `javascript npm run start`
